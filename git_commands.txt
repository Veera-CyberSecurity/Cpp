-> git clone <GitHub_repo_link> // to get source from gitHub
-> git pull origin //to get the updated source from github
-> git branch // to show the list of branches present in the project source
-> git status // to show the modified filed in present branch
-> git add <file name> // to add the modified files into branch
-> git commit -m "<file_modified_message/description>" // to give the modified file description
-> git push origin <branch name> // To push local changes into central
-> git log //to show the commit id's / commit msg's for the modified source code changes
-> git show <commit_id> // to show modified file changes
-> git diff <path_of_the_file/file_along_eith_path/file_name> // to show diff of the file before commit&push the changes
